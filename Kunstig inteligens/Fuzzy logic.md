#### Fuzzy and Crisp sets
Indicator function for a crisp set A:
$$
I_{A}(x)\rightarrow\{0,1\}
$$
Membership function for a fuzzy set A:
$$
\mu_{A}(x)\rightarrow [0,1]
$$
